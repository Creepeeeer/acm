**中国剩余定理**

给定n个同余方程

$$\left\{ \begin{aligned} x\equiv r1(mod m_1)\\x \equiv r2(mod m_2)\\....\\x \equiv r_n(modm_n) \end{aligned}\right.$$

并且$m_i,m_j$两两互质，求满足同余方程的最小正数解



方法：

将x分为n块，即$x=c_1+c_2+.....c_n$

令m=$\prod_{i=1}^nm_i$,由于$m_i,m_j$两两互素，其实m也表示m_i的lcm

且如果$i!=j ,c_i mod m_i=r_i, c_i mod m_j =0$

令$a_i=\frac{m}{m_i},inva_i为a_i在modm_i下的逆元$

$c_i=r_i*a_i*inva_i$

所有整数解可以表示为$x+km$,那么最小正数解为x%m

```c++
void exgcd(ll a, ll b, ll& x, ll& y) {
	if (b == 0)
	{
		x = 1, y = 0;
		return;
	}
	exgcd(b, a % b, y, x);
	y -= (__int128)(a / b)* x;
}
pll crt(vector<pll>&ve){//first mod second r_i
	ll m=1;
	for(auto&[x,y]:ve){
		m*=x;
	}
	ll an=0;
	for(auto&[mod,r]:ve){
		ll te=m/mod;
		ll x,y;
		exgcd(te,mod,x,y);
		x=((x%m)+m)%m;
		an+=(__int128)r*x%m*te%m;
		an%=m;
	}
	an=(an%m+m)%m;
	return pair(an,m);
}
```
