**正常情况n个方程解n个数，系数得到的是一个n*(n+1)的矩阵**

**如果表达式冗余就补充变量（系数为0），表达式不足就补充表达式**

解分为**唯一解，矛盾，多解**

第i行解的就是$x_i$的值，如果此时系数为1称其为主元，否则为自由元

**流程：**

假设此时要解第i个变量，那么从$[1,i-1]$的自由元行（即$a[j][j]==0$)

和$[i,n]$行中选出第i列系数最大的行，如果最大系数都是0那么$x_i$是自由元，否则将这行与i行swap，将这行每个数$/a[i][i]$

，将所有非i行元素$a[k][j]-a[i][j]*rate$

- 唯一解：无自由元
- 矛盾：有自由元，且至少有一个自由元的n+1列非0
- 多解：有自由元，且所有自由元n+1列为0

**主元可能依赖其他自由元，并且如果主元是$x_i$那么自由元的序号一定>i，但不会依赖其他主元，自由元不依赖其他主元和自由元**

```c++
const int N=100;
double a[N+1][N+2];
const double eps=1e-6;
int gaussadd(int n){//1唯一解 0多解 -1无解
	for(int i=1;i<=n;i++){
		int ma=i;
		for(int j=1;j<=n;j++){
			if(j<i&&abs(a[j][j]>=eps))continue;
			if(abs(a[j][i])>abs(a[ma][i]))ma=j;
		}
		swap(a[i],a[ma]);
		if(abs(a[i][i])>=eps){
			double te=a[i][i];
			for(int j=i;j<=n+1;j++)a[i][j]/=te;
			for(int j=1;j<=n;j++){
				if(i==j)continue;
				double rate=a[j][i]/a[i][i];
				for(int k=i;k<=n+1;k++){
					a[j][k]-=rate*a[i][k];
				}
			}
		}
	}
	bool ok=false;
	for(int i=1;i<=n;i++){
		if(abs(a[i][i])<eps){
			if(abs(a[i][n+1])>=eps)return -1;
			ok=true;
		}
	}
	return ok?0:1;
}
```



**时间复杂度$O(n^3)$**
