### **裴蜀定理：**

如果a,b是不全为0的整数，那么有整数x,y 使$ax+by=gcd(a,b)$

**重要理解**

a和b的$gcd=ax+by$ 随意给定整数a,b，能得到的最小正数差值

如果a,b不全为0的整数，并且$ax+by=c$有解，那么c一定是gcd(a,b)的整数倍

a,b两项的裴蜀定理可以推广到多项的情况



### 拓展欧几里得:

解$ax+by=gcd(a,b)$的一组特解

$ax+by=bx'+(a-a/b*b)y' =ay'+b(x'-ay'/b)$

```c++
void exgcd(ll a, ll b, ll& x, ll& y) {
	if (b == 0)
	{
		x = 1, y = 0;
		return;
	}
	exgcd(b, a % b, y, x);
	y -= (a / b) * x;
}
```



#### 求逆元

$ax \equiv 1(mod b)$

费马小定理只能求b为素数的情况，**拓展欧几里得可以求a,b互素的情况**

$ax-by=1$
