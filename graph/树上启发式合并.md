树上启发式合并的特征：

1，没有修改操作

2，可以通过遍历子树，建立信息统计，得到所有查询的答案

树上启发式合并的过程

void dfs(u, keep)，u表示当前节点，keep表示是否保留子树u对信息的贡献

1，先遍历所有轻儿子的子树，遍历结束时，消除对信息的贡献，dfs(轻儿子，0)

2，再遍历唯一重儿子的子树，遍历结束时，保留对信息的贡献，dfs(重儿子，1)

3，考察单个节点u，对信息进行贡献

4，再遍历所有轻儿子的子树，上面的每个节点，都重新对信息进行贡献

5，得到子树u的答案

6，如果keep == 0，消除子树u的贡献，如果keep == 1，保留子树u的贡献

```cpp
    auto dfs=[&](auto&&self,int u,int f)->void{
    	siz[u]=1;
    	for(auto v:g[u]){
    		if(v==f)continue;
    		self(self,v,u);
    		siz[u]+=siz[v];
    		if(siz[v]>siz[son[u]]){
    			son[u]=v;
    		}
    	}
    };
    dfs(dfs,1,0);
    auto add=[&](auto&&self,int u,int f)->void{
    	if(++ccnt[c[u]]==1)diff++;
    	for(auto&v:g[u]){
    		if(v==f)continue;
    		self(self,v,u);
    	}
    };
    auto del=[&](auto&&self,int u,int f)->void{
    	if(--ccnt[c[u]]==0)diff--;
    	for(auto&v:g[u]){
    		if(v==f)continue;
    		self(self,v,u);
    	}
    };
    auto dfs2=[&](auto&&self,int u,int f,bool kep)->void{
    	for(auto&v:g[u]){
    		if(v==f||v==son[u])continue;
    		self(self,v,u,0);
    	}
    	if(son[u]!=0)self(self,son[u],u,1);
    	if(++ccnt[c[u]]==1)diff++;
    	for(auto&v:g[u]){
    		if(v==f||v==son[u])continue;
    		add(add,v,u);
    	}
    	an[u]=diff;
    	if(!kep)del(del,u,f);
    };
    dfs2(dfs2,1,0,0);
```
