**最大匹配**是指在二分图中找到一组边，使得这些边没有公共顶点，且边的数量最大。通过寻找**增广路径**，可以逐步优化匹配，直到达到最大匹配。

增广路径的特点是：

1. 它是交替路径，路径上的边交替属于匹配和非匹配。
2. 它的两个端点都不属于当前匹配。
3. 通过对增广路径进行取反操作，可以得到更大的匹配。

匈牙利算法步骤

1. 初始化匹配集为空。
2. 遍历二分图中的每个顶点，尝试为其找到增广路径。
3. 如果找到增广路径，则更新匹配集。
4. 重复上述过程，直到无法找到新的增广路径。




**dfs和bfs写法时间复杂度都是$O(n^2)$**

**dfs写法1**

```c++
bool vis[N+1];
int matchr[N+1],matchl[N+1];
    int an=0;
    auto dfs=[](this auto&&self,int u)->bool{
    	for(auto&v:g[u]){
    		if(vis[v])continue;
    		vis[v]=true;
    		if(!matchr[v]||self(matchr[v])){
    			matchr[v]=u;
                matchl[u]=v;
    			return true;
    		}
    	}
    	return false;
    };
    for(int i=1;i<=n;i++){
    	fill(vis,vis+m+1,false);
    	if(!matchl[i]&&dfs(i))an++;
    }
    cout<<an;
    return 0;
}
```

下面这种写法其实等效上面这种写法，如果一个左端点能作为右端点的match那么它一定是作为开头被dfs过的

**bfs写法**

```c++
vector<int>g[N+1];
int matchl[N+1],matchr[N+1],prer[N+1];
bool visl[N+1],visr[N+1];
    int an=0;
    auto bfs=[&](int i)->bool{
        queue<int>qu;
        fill(visl,visl+n+1,false);
        fill(visr,visr+m+1,false);
        fill(prer,prer+m+1,0);
        qu.push(i);
        visl[i]=true;
        while(!qu.empty()){
            int u=qu.front();
            qu.pop();
            for(auto&v:g[u]){
                if(visr[v])continue;
                visr[v]=true;
                prer[v]=u;
                if(!matchr[v]){
                    int vv=v;
                    while(vv){
                        int uu=prer[vv];
                        int nex=matchl[uu];
                        matchl[uu]=vv;
                        matchr[vv]=uu;
                        vv=nex;
                    }
                    return true;
                }
                else{
                    auto&&uu=matchr[v];
                    if(!visl[uu]){
                        visl[uu]=true;
                        qu.push(uu);
                    }
                }
            }
        }
        return false;
    };
    for(int i=1;i<=n;i++){
        if(!matchl[i]&&bfs(i))an++;
    }
    cout<<an;
```
