## 比赛对拍

新建3个文件

**my.cpp**

**good.cpp**（暴力）

**generator.cpp**(数据生成器)

新建对拍脚本 duipai.sh

```sh
#!/bin/bash

while true; do
    ./generator > in.txt

    ./good < in.txt > good.out
    ./my   < in.txt > my.out

    if ! diff -q good.out my.out > /dev/null; then
        echo "input"
        cat in.txt
        echo "correct"
        cat good.out
        echo "wrong"
        cat my.out
        break
    fi
done

```



在终端运行一次

```c++
g++ generator.cpp -O2 -std=c++17 -o generator
g++ good.cpp      -O2 -std=c++17 -o good
g++ my.cpp        -O2 -std=c++17 -o my
chmod +x duipai.sh 
```



然后对拍就输入

```c++
./duipai.sh
```
