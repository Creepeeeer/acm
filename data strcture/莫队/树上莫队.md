树上的某些链上查询信息不能被树剖+线段树解决，那么可以尝试树上莫队

括号序：dfs过程中，刚到节点x时，给x分配开始序，遍历完x所有子树要离开的时候给x分配结束序seg

那么此时一个点有两个序号，开始序st[u]和结束序en[u]

**做法**

路径类型1:u或v就是lca(u,v) 那么路径转化为seg[st[u].....st[v]]

​	       2:u或v不是lca(u,v)那么路径转化为seg[en[u].....st[v]]

```c++
for (int i = 0; i < m; i++) {
		int x, y;
		cin >> x >> y;
    	if(sa[x]>sa[y])swap(x,y);
		int z = lca(x, y);
		if (z == x) {
			q[i] = {sa[x], sa[y], -1, i};
		}
		else q[i] = {en[x], sa[y], z, i};
	}
```

**某个节点在转化后的序列中，出现一次认为是生效，出现两次认为是无效**

每条路径都可以转化为括号序列中连续的一段，然后可以利用各种莫队算法求解
